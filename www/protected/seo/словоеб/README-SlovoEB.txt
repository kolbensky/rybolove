СловоЁБ 2.1 - http://seom.info/2012/05/04/slovoeb-2-0/

СловоЁБ - это урезанная бесплатная версия программы Key Collector ( http://www.key-collector.ru/ ). Программа имеет ряд технических ограничений, обладает только базовыми возможностями и упрощенной системой загрузки страниц и обработки ошибок.

Программа позволяет собирать данные с сервиса Яндекс.Wordstat, а также поисковые подсказки с популярных поисковых систем. Также вы можете снимать конкуренцию по запросам в поисковых системах Яндекс и Google. По каждому слову можно определить релевантные страницы на сайте (по версии ПС Яндекс и Google).

СловоЁБ может работать с данными в многопоточном режиме (ограничено 3 потоками). Работает с сервисом распознавания капчи antigate.com


Установка программы СловоЁБ.

1. Скачайте архив с программой по ссылке: http://seom.info/new/SlovoEB/
2. Убедитесь, что архив является разблокированным. Для этого в свойствах файла (в контекстном меню выберите пункт "Свойства") нажмите кнопку "Разблокировать" / "Unblock", если такая присутствует.
3. Разархивируйте содержимое архива.

Активация и запуск программы СловоЁБ.
1. Запустите исполняемый файл Slovoeb.exe.

Обновления программы.
1. Обновления могут быть нерегулярными, в отличии от Key Collector.
2. Обновления вы скачиваете самостоятельно, проверяя их наличие на странице http://seom.info/tools/

Важная информация!
1. Программа требует установленной полной версии Microsoft .NET Framework 4.0 FULL. Как правило, по умолчанию в системе либо установлена версия 3.5, либо ограниченная клиентская версия. Загрузить полную версию Microsoft .NET Framework 4 Вы можете по ссылке: http://www.microsoft.com/ru-ru/download/details.aspx?id=17718
Если с установленным 4.0 Full программма не запускается, то попробуйте удалить его и установить Microsoft .NET Framework 4.5 Full:
http://www.microsoft.com/ru-ru/download/details.aspx?id=30653

3. Убедитесь, что при работе программа обладает достаточными правами в системе. Симптомом недостатка прав зачастую является невозможность без ошибок создать новый или открыть существующий проект. Для диагностики проблемы запустите процесс Slovoeb.exe из под администратора (не просто из под учетной записи пользователя по умолчанию, а через контекстное меню - "Запуск от имени администратора"). Далее попробуйте создать новый проект на общедоступном месте, например, на рабочем столе. Если ошибки при создании проекта исчезли, значит, дело было в недостатке прав.
